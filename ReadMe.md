# このソフトウェアについて #

HelloMouse20160702は私個人が学習目的で作成したソフトウェアです。

C++/win32にてマウスを使ってみました。
クリックした位置に座標を表示します。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
